# HospitalSite

Сайт больницы, разработанный в рамках конкурса **«ПрофМастерство»**  

## Стек

- HTML5
- CSS3
- JavaScript

## Структура проекта

```text
HospitalSite/
├── assets/
│   ├── css/        # стили (базовые + тема)
│   ├── js/         # логика интерфейса
│   └── images/     # изображения проекта
├── pages/          # отдельные страницы 
├── sections/       # секции главной страницы
└── index.html      # главная страница
```
